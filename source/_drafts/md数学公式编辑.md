- 方法1：\frac {分子}{分母}

|           书写            |      Markdown       |
| :-----------------------: | :-----------------: |
| ```$$\frac{a+b}{c+d}$$``` | $$\frac{a+b}{c+d}$$ |
|    ```$$b \over a$$```    |    $$b \over a$$    |
|     ```$$\frac12$$```     |     $$\frac12$$     |
